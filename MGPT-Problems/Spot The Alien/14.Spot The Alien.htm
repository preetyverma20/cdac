MGPT Problem 14

Spot The Alien

NCST has decided to start a new center on a small blue-green planet, in a galaxy far, far away. Humans have colonized this planet for some years now, but there are some aliens on the planet as well. NCST's courses are, however, available only for humans. The aliens look exactly like humans and can be differentiated only on the basis of DNA testing. Given a set of life forms, you have to spot the alien.

The DNA of each life form can consist of only 4 components: A, C, T and G. The DNA string of each life form is given. Based on the DNA string of each life form, its Similarity with every other life form is computed.
Calculation of Similarity is done using the following rules:

  1. If any component is present at the same place, in two DNA strings, the Similarity count goes up by 3 points.
  2. If any component is present in both DNA strings, but not in the same place, the count goes up by 1 point. Any component already matched in the previous rule cannot be considered here.
  3. If any matching pairs of components are present in both DNA strings, the count goes up by 2 points.
  4. A pair once used for matching cannot be used again to match another part of the DNA string.

For example: 

       X's DNA string : ATGCAT
       Y's DNA string : GATCGG
  • According to rule no. 1, only C is in the same place in both strings and the count = 3.
  • According to rule no. 2, A, T and G are present in the strings, but not at the same place, hence the count increases by 1, for each component. The count now is 6 (3 + 1 + 1 + 1).
  • According to rule no.3, AT forms a pair present in both strings, and hence the count goes up by 2. The count now is 8 (6 + 2). There are two pairs of AT in X's DNA, but only one in Y's DNA, hence it can be counted only once.
  • X's total similarity with Y is 8.

NOTE:

  • AT and TA cannot be counted as a valid pair, i.e., both strings must have the paired components in the same sequence.
  • A sequence like ATG in a DNA can be eligible for two pairings, i.e. AT and TG.

For detecting the alien, NCST has devised a clever system called the Similarity Measure. Each life form's Similarity Measure is the summation of it's Similarity with every other life form, i.e. If there are 3 life forms: X, Y and Z, then X's Similarity Measure is the sum of it's Similarity values with Y and Z. The life form with the least Similarity Measure is the alien. It is guaranteed that there will be only one life form with the least Similarity Measure.
In the example considered X calculates its similarity with every other life form given and thus calculates its Similarity Measure as a sum of all these similarity values.

Input specification:

  • First line contains a pair of integers N and L, indicating the number of individuals and length of DNA string, respectively. N will always be greater than 2. L will always be greater than 0.
  • The next N lines contain a character and a string each, specifying the ID of the individual and it's DNA string. The ID and the DNA string are separated by one space.

Output specification:

The ID of the individual who is the alien.


Sample Input and Output:


Input:
3 4
X ATGC
Y GATC
Z AAGC

Output:
Y

Input:
3 6
X AAAATC
Y TCAGGA
Z CTGAGA 

Output:
X
